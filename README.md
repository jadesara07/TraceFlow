# TraceFlow

Observability-ready Node.js backend project.

## Features

- Express backend API
- file-based database
- Logging system
- Metrics collection
- Trace collection
- Middleware lifecycle tracking
- Async request handling
- Error monitoring

## Tech Stack 

- Node.js
- Express.js
- File System APIs

## Observability Features

- logs.txt
- metrics.json
- traces.json
